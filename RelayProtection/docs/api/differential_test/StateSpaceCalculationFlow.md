# 差动保护底层解算算法：状态空间方程模型

在继电保护测试仪的底层发波逻辑中，我们采用**状态空间方程 (State-Space Equation)** 的思想，将“用户测试意图”与“保护装置内部算法”进行完美的数学剥离。通过建立目标向量与物理约束的方程组，实现逻辑自洽的终极解算引擎。

---

## 第一步：确立“内部目标向量”

无论外部变压器接线组别如何，首先根据“测试相别”，生成保护装置内部必须算出的目标比对电流向量（假设高压侧基准为 $0^\circ$，低压侧基准为 $\theta_2$）：

### 1. A相差动（单相目标）
* **高压侧内部**：$I_A' = i_1 \angle 0^\circ, \quad I_B' = 0, \quad I_C' = 0$
* **低压侧内部**：$I_x' = i_2 \angle \theta_2, \quad I_y' = 0, \quad I_z' = 0$

### 2. AB相差动（两相目标）
* **高压侧内部**：$I_A' = i_1 \angle 0^\circ, \quad I_B' = i_1 \angle 180^\circ, \quad I_C' = 0$
* **低压侧内部**：$I_x' = i_2 \angle \theta_2, \quad I_y' = i_2 \angle (\theta_2 + 180^\circ), \quad I_z' = 0$

### 3. ABC三相差动（三相平衡目标）
* **高压侧内部**：$I_A' = i_1 \angle 0^\circ, \quad I_B' = i_1 \angle -120^\circ, \quad I_C' = i_1 \angle 120^\circ$
* **低压侧内部**：$I_x' = i_2 \angle \theta_2, \quad I_y' = i_2 \angle (\theta_2 - 120^\circ), \quad I_z' = i_2 \angle (\theta_2 + 120^\circ)$

---

## 第二步：建立“物理发波方程组”

将物理通道的输出设为未知数：高压侧 $[I_A, I_B, I_C]$，低压侧 $[I_x, I_y, I_z]$。
根据保护装置的**校正方式**，物理电流必须严格满足以下线性代数方程组之一：

### 场景 1：无校正 (None)
保护装置不做校正，测试仪直接发波，但必须人工叠加变压器钟点数（Clock）造成的物理角差。
* **高压侧方程**：
  $$I_A = I_A', \quad I_B = I_B', \quad I_C = I_C'$$
* **低压侧方程**（设 $\Delta\theta = (12 - Clock) \times 30^\circ$）：
  $$I_x = I_x' \cdot e^{j\Delta\theta}, \quad I_y = I_y' \cdot e^{j\Delta\theta}, \quad I_z = I_z' \cdot e^{j\Delta\theta}$$

> **注**：这就是为什么在无校正时会看到低压侧角度整体偏移，或者在特定的钟点数下触发端子的“相序轮换”。

### 场景 2：Y 侧零序校正 (且无相位矩阵)
保护装置内部强行减去了零序电流 $I_0 = \frac{I_A + I_B + I_C}{3}$。
* **高压侧方程组**：
  $$I_A - \frac{I_A + I_B + I_C}{3} = I_A'$$
  $$I_B - \frac{I_A + I_B + I_C}{3} = I_B'$$
  $$I_C - \frac{I_A + I_B + I_C}{3} = I_C'$$

> **注**：这是一个欠定方程组。为了**最小化相电流峰值**（即“最小化单相电流”的真正含义：$\min \max(|I_A|, |I_B|, |I_C|)$，防止功放过载溢出），最经典的解法是采用**双通道对消**：
> 令 $I_C = 0, I_B = -I_A$，代入得 $I_A - \frac{I_A - I_A + 0}{3} = I_A \implies I_A = I_A'$。
> 即 $I_A = i_1, I_B = -i_1, I_C = 0$。此解的峰值为 $i_1$，远小于单通道发波的 $1.5 i_1$。

### 场景 3：Y 侧相位校正 (Y-side)
保护装置执行了线电流相减，并默认除以 $\sqrt{3}$ 归一化（国内主流保护标准算法，自动消除零序）。
* **高压侧方程组**：
  $$\frac{I_A - I_B}{\sqrt{3}} = I_A'$$
  $$\frac{I_B - I_C}{\sqrt{3}} = I_B'$$
  $$\frac{I_C - I_A}{\sqrt{3}} = I_C'$$

> **注**：代入 A 相单相目标 $[i_1, 0, 0]$ 时，根据**最小化相电流峰值**的约束策略：
> 要满足 $\frac{I_A - I_B}{\sqrt{3}} = i_1$ 即 $I_A - I_B = \sqrt{3} i_1$，为使 $\max(|I_A|, |I_B|)$ 最小，应当使两相完全对称分担：
> 取 $I_A = \frac{\sqrt{3}}{2} i_1 \approx 0.866 i_1$， $I_B = -\frac{\sqrt{3}}{2} i_1 \approx -0.866 i_1$，$I_C = 0$。
> 这也是双通道发波。相比于单通道发波（$I_A = \sqrt{3}i_1 \approx 1.732 i_1$），该方法将电流峰值降低了一半，完美避免了仪器输出过载！

### 场景 4：D 侧相位校正 (Delta-side)
与 Y 侧同理，但方向反转，作用于低压侧方程。
* **低压侧方程组**：
  $$\frac{I_x - I_z}{\sqrt{3}} = I_x'$$
  $$\frac{I_y - I_x}{\sqrt{3}} = I_y'$$
  $$\frac{I_z - I_y}{\sqrt{3}} = I_z'$$

> **注**：同样适用“最小化相电流峰值”约束，采用双通道对消（如 $I_x = \frac{\sqrt{3}}{2} i_1, I_z = -\frac{\sqrt{3}}{2} i_1$）。

---

## 上位机解算架构工作流（终极形态）

利用上述状态空间方程，上位机（测试软件）的底层发波计算流如下：
1. **获取目标**：先看“测什么”（A、AB、ABC），生成右侧的目标内部向量 $[I_A', I_B', I_C']$。
2. **构建方程**：再看“开了什么校正”，选中对应的左侧方程组场景。
3. **附加约束（核心规则）**：强制应用**“最小化相电流峰值 (Minimax)”**的功放约束策略。为了防止单相过载（最小化单相电流），对于欠定方程，求解使 $\max(|I_A|, |I_B|, |I_C|)$ 最小的解，这通常表现为**双通道对消**法。
4. **矩阵求解**：将右边目标值代入左边方程组，结合最小化峰值约束求解出 $[I_A, I_B, I_C]$ 和 $[I_x, I_y, I_z]$，最终下发给硬件发波。

这套严密的数学框架确保了无论是何种变压器接线，还是任何奇特的测试要求，底层算法都永远保持逻辑自洽，彻底消除了基于经验硬编码 (Hardcoding) 带来的兼容性与边界漏洞。
